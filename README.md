# Projeto Web Scraping

## Projeto de Scraping extraindo dados de vagas para o Power BI

Esse Projeto de Web Scraping é pra trazer as vagas de empregos de três aplicativo de empregos, que são: Linkedin, Glassdoor e Indeed, depois irei transformar em BI as vagas de analista de dados/BI

Começaremos na estruturação do scrapy para 3 sites, então iremos criar 3 spider para que possamos fazer o parser corretamente sem interferencia do outro site.
